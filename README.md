# Команди управління

### Запуск програми
Є два способи запустити програму:
1. **Через термінал** (запускає веб-сервер на порту 5000):
   `dotnet run --urls "http://localhost:5000"`

2. **Запуск через файл**:
   Запустіть `TournamentPlatform.exe` у папці `bin/Release`.

### Дані для входу (Тестові)
Для перевірки функціоналу використовуйте ці дані:
- **👑 Адміністратор**: логін `admin`, пароль `123`
- **⚖️ Журі**: логін `jury`, пароль `123`
- **👥 Команди**: логін — назва команди, пароль за замовчуванням `123`

### Повне скидання програми
Виконує повний «зброс» усіх даних (видаляє базу даних):
`rm tournaments.db*`
