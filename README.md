# Команди управління

### Запуск програми
Є два способи запустити програму:
1. **Через термінал** (запускає веб-сервер на порту 5000):
   `dotnet run --urls "http://localhost:5000"`

2. **Запуск через exe-файл**:
   Зберіть реліз командою: `dotnet publish -c Release -o ./publish`
   Потім запустіть `TournamentPlatform.exe` з папки `publish\`.

   ⚠️ **Важливо**: запускати exe потрібно з папки, де він знаходиться — разом з усіма файлами (включно з папкою `wwwroot\`). Не переміщуйте тільки exe окремо.

### Повне скидання програми
Виконує повний "зброс" даних, які були зроблені у програмі (видаляє базу даних):
`rm tournaments.db*`

- **Адміністратор**: `admin@role.com` / `admin123`
