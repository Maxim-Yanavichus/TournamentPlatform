# Команди управління

### Запуск програми
Є два способи запустити програму:
1. **Через термінал** (запускає веб-сервер на порту 5000):
   `dotnet run --urls "http://localhost:5000"`

2. **Запуск через файл**:
   Запустіть `TournamentPlatform.exe` у папці `bin/Release`.

### Повне скидання програми
Виконує повний "зброс" даних, які були зроблені у програмі (видаляє базу даних):
`rm tournaments.db*`
